Daily Story

Read a new story every day in the language you're learning, at your skill level.

Node.js Express Claude AI
View Live Site GitHub Repository

Overview

Language immersion doesn't require a plane ticket. Daily Story delivers fresh, AI-generated stories in eight languages across four skill levels—beginner to advanced. Each story comes with comprehension quizzes to reinforce what you've learned and track your progress.

Whether you're brushing up on Spanish before a trip or diving deep into Japanese grammar, Daily Story gives you something new to read every single day, perfectly calibrated to challenge you without overwhelming you.

The Challenge

One of the best ways to learn a language is to immerse yourself in it through reading. The problem? Finding content that matches your exact skill level is difficult. Too easy and you're not learning; too hard and you're frustrated.

I found myself constantly asking ChatGPT to generate short stories at my skill level for language practice. It worked, but it was tedious to do manually every day. I realized this could be automated into a daily habit-building tool that others could benefit from too.

The Solution

Building Daily Story taught me that working with AI is less about the code and more about the conversation. The real challenge wasn't the architecture—it was prompt engineering.

After a few days of testing, I noticed the AI kept generating eerily similar stories. Without memory of what it had created before, the model naturally gravitated toward the same narrative patterns. The solution? A curated library of about 100 themes per skill level that the app randomly selects from before each generation.

This wasn't just about variety for variety's sake. A theme like "navigating cultural misunderstandings" works beautifully for intermediate learners but falls flat for beginners who need simpler vocabulary. Each proficiency level needed its own theme collection, carefully chosen to match both linguistic complexity and conceptual accessibility.

Key Features

Fresh Content Daily

Every day brings a new story perfectly calibrated to your current proficiency—challenging enough to push your skills forward, accessible enough to keep you engaged.

Active Comprehension

Three targeted questions after each story transform passive reading into active learning, helping you retain vocabulary and internalize grammar patterns.

8 Languages, 4 Levels

From beginner Spanish to advanced Japanese, find your perfect match across eight languages and four proficiency tiers.

Technical Implementation

The architecture is intentionally straightforward. Built with Node.js and Express, the app runs on a simple cron schedule that handles AI content generation efficiently and cost-effectively.

Every day at midnight, the server submits a batch request to the Claude API for 32 different stories—8 languages at 4 skill levels each. Batch requests process at half price but can take up to 24 hours, so at 6 PM the next day, the app retrieves the completed stories and stores them in a JSON file. When users visit, the appropriate story is rendered from this cache.

Running on Google Cloud Run means I only pay for active compute time. By design, the app goes idle after sending the batch request and only wakes up 18 hours later to retrieve results—eliminating expensive wait time while the API processes stories. No database overhead, no complex state management—just reliable, daily language practice delivered through a clean API.

What I Learned

This project deepened my understanding of the Claude API and the nuanced art of prompt engineering. I learned that quality matters more than quantity—stories need to be genuinely engaging and leave learners feeling measurably better at their target language.

The technical challenge came in structuring prompts to return consistent JSON across eight languages, ensuring that titles, quiz questions, and story content could be reliably parsed despite the linguistic variety. It required more iteration than I expected, but the result is a robust system that handles multilingual content seamlessly.
