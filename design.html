Design Philosophy

Visual Distinction

Project cards: Playful (tilted), technical showcase

Functional Distinction

Project cards: Action-oriented (demo, GitHub links)

Consistency

Badges & Tags

Small labels for status and categorization

Badges vs Tags: When to Use Each

Badges

Purpose: Celebrate achievements, status, or milestones

Answer the question:

"What have you achieved or earned?"

Characteristics:

  • Convey accomplishment or special designation
  • Often quantifiable or measurable
  • Emotional tone: Pride, accomplishment
  • More visually prominent/colorful

Examples:

140+ Hours PPL Certified Coast to Coast High Performance

Tags

Purpose: Categorize, label, or organize information

Answer the question:

"What category or type is this?"

Characteristics:

  • Factual classification or grouping
  • Help with filtering or searching
  • Emotional tone: Neutral, informational
  • More subtle, less prominent styling

Examples:

JavaScript React Node.js PostgreSQL

Usage Guidelines

‚úì Use Badges For:

Aviation hours, certifications, hiking achievements, proficiency levels, milestone numbers

‚úì Use Tags For:

Tech stacks, programming languages, project categories, skills, frameworks, tools

Design Principles

1. Clarity First

Information hierarchy guides visitors naturally. Every element has a purpose.

2. Technical Craft

Clean, intentional design demonstrates attention to detail and engineering mindset.

3. Subtle Motion

Purposeful animations enhance user experience without distraction.

4. Authentic Voice

Personal touches and genuine interests make the site memorable.

Implementation Notes

Using the Design System

  1. Colors: Use CSS custom properties (e.g., var(--color-primary)) for consistency and easy theme updates
  2. Spacing: Follow the 4px base unit system for consistent rhythm
  3. Typography: Use semantic heading levels (h1-h6) and let the styles handle the sizing
  4. Components: Combine base classes with modifiers (e.g., .btn.btn-primary.btn-lg)
  5. Responsive: Design system includes mobile breakpoints - test on all screen sizes
  6. Performance: All animations use transform and opacity for 60fps performance
